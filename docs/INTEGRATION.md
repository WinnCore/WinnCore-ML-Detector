# Integration with WinnCoreAV

How to use trained ML models in WinnCoreAV for hybrid detection.

## Quick Integration

1. Train model in this repo
2. Copy ONNX file to WinnCoreAV
3. WinnCoreAV loads model at startup
4. Run inference during file scans

[Full guide at https://github.com/WinnCore/WinnCore-ML-Detector/wiki/Integration]
